# Claude Crash Course

## Setup
1) Create and activate venv:
   python3 -m venv .venv
   source .venv/bin/activate

## Run scripts
python3 scripts/hello.py

## Run agent
1) Put .txt files in ops_agent/inputs/
2) Run:
   python3 ops_agent/main.py
3) Output is in ops_agent/outputs/report.md
